**Qwen Cloud Global AI Hackathon — Track: MemoryAgent**

## Endpoint Used

```
https://dashscope-intl.aliyuncs.com/compatible-mode/v1
```

This is the official Alibaba Cloud DashScope international endpoint, exposed in an OpenAI-compatible format and used by all three agents (`memory_agent.py`, `bi_agent.py`, `action_agent.py`) via the `openai` Python SDK:

```python
client = OpenAI(
    api_key=os.environ.get("DASHSCOPE_API_KEY"),
    base_url="https://dashscope-intl.aliyuncs.com/compatible-mode/v1"
)
```
